Docker Basic Commands

Let's learn basic necessary commands in "Docker"

To check whether "Docker" has been installed or not:

"docker run hello-world"

To check all the docker images:

"docker images" or "docker image ls"

To check all the running and existing containers:

"docker ps -a". Here, "-a" means "all".

To check only running containers:

"docker ps"

To remove a running image:

"docker rmi -f docker_image_id"

To remove stopped image:

"docker image rm docker_image_id"

To stop running container:

"docker kill container_id"

To run a container:

"docker run container_id"

To renaming a existing a container:

"docker rename old-name new-name"
