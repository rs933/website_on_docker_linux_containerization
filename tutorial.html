Docker Tutorial

How To Set Up A Web Server In "Docker"?

Let's learn how to setup apache server in "Docker"

  • Step-1:

    Create a diroctory “public_html”(you can put any name of this directory) and inside of thise directory create a simple html file called “index.html”

  • Step-2:

    Create a docker file inside the “public_html” directory. Remember the name of the docker file should be “Dockerfile” without any extension. And paste this code in “Docker file”:
    FROM httpd:2.4
    COPY . /usr/local/apache2/htdocs/

  • Step-3:

    Create a docker image using this command:
    “docker build –t name_of_image:tag directory_location”

  • Step-4:

    Finally, run this command to run your apache server:
    "docker run -p 80:80 --name name_of_image-1 name_of_image"

Finally, you can see the index file on localhost on your browser.

Let's learn how to setup nginx server in "Docker"

  • Step-1:

    Create a diroctory “public_html”(you can put any name of this directory) and inside of thise directory create a simple html file called “index.html”

  • Step-2:

    Create a docker file inside the “public_html” directory. Remember the name of the docker file should be “Dockerfile” without any extension. And paste this code in “Docker file”:
    FROM nginx:alpine
    COPY . /usr/share/nginx/html

  • Step-3:

    Create a docker image using this command:
    "docker build -t image_name:tag ."

  • Step-4:

    Run this command to check whether the has been created or not:
    "docker images"

  • Step-5:

    Finally, run this command to run your apache server:
    "docker run -d -p 80:80 image_name:tag_name"

Finally, you can see the index file on localhost on your browser.
